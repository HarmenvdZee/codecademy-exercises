CSS selectors and specificity

Which styling rule is applied when multiple rules collide

On this page we will discuss how the browser determines which styling to apply when their are multiple styling rules acting on the same element and property of that element.

This is mainly done by looking at the which selector is used for the styling rules of that element. Based on the selectors the browser creates an order of importance for the styling rules.

The order of importance for these selectors will be covered in the following table

Table of selector importance

Selector HTML CSS
ID id='IDselector' #IDselector {}
class class='classSelector' .classSelector {}
type <type> </type> type {}
no value all elements * {}

So for example if an elements font size is determined by both a styling rule that uses its type and a styling rule that uses its ID. Then it will take the font size of the rule that uses the ID.
